# Видео: приватный диалог по мэтчу — как выглядит и как в него попасть

## Что показать в видосе

### 1. Что такое приватный диалог

Это **чат между кандидатом и работодателем** по конкретному мэтчу. Сообщения видят только эти двое; в WordPress админ видит переписку в карточке мэтча (метабокс «Chat (this match)»).

### 2. Как попасть в чат

**Соискатель (кандидат):**
- Зайти в приложение → **Club services** → **HR service** → **My matches**.
- В списке мэтчей нажать **Chat** у нужного мэтча.
- Откроется экран чата с работодателем по этой вакансии. Можно писать сообщения, нажимать Send.

**Работодатель:**
- Зайти в приложение → **Club services** → **HR service** → **Opened resumes** (или **Candidates**, если мэтч ещё ждёт одобрения; после одобрения он попадёт в Opened resumes).
- В списке нажать **Chat** у нужного кандидата/мэтча.
- Откроется тот же чат — переписка общая для этого мэтча.

У каждого мэтча **один общий чат**: все сообщения привязаны к мэтчу и отображаются и кандидату, и работодателю.

### 3. Что показать в кадре

1. **Экран Club services** — раскрытый блок **HR service**, под ним пункты (Open vacancies, Place candidacy, My matches и т.д.).
2. **Соискатель:** заход в My matches → нажатие **Chat** у мэтча → экран чата (лента сообщений, поле ввода, Send).
3. **Работодатель:** заход в Opened resumes → нажатие **Chat** у строки с кандидатом → тот же чат (те же сообщения).
4. При желании: в админке WordPress открыть запись типа **Matches (HR)** → выбрать мэтч → внизу метабокс **«Chat (this match)»** с историей переписки.

### 4. Краткий текст для озвучки (англ.)

- "Private chat is available only after a match exists between you and the other side. As a job seeker, go to Club services → HR service → My matches, then tap Chat on a match. As an employer, go to Opened resumes and tap Chat next to the candidate. The same conversation is visible to both. Messages are stored on our server and can be reviewed in WordPress admin under the match."
